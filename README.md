# EvolutionAI

Interactive evolutionary theory simulator.

[![Live Demo](https://img.shields.io/badge/demo-evolution--ai--six.vercel.app-2d7d4a)](https://evolution-ai-six.vercel.app)

## Overview

A browser-based visual simulator for exploring core ideas from evolutionary biology — selection pressure, mutation rate, population dynamics — through live, manipulable parameters.

## Design & Tech

- **Stack:** Single-page site, vanilla JavaScript, Canvas
- **Deployment:** Vercel
